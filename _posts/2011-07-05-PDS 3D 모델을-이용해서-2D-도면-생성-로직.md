PDS 3D 모델을 이용해서 2D 도면 생성 로직을 간단하게 FLOW CHART형식으로 표현해 보았습니다.\
여기서 2D DWG FILE NAME은 PDS에서 작성한 HIDDEN 도면을 말하는 겁니다.\
그리고 121,122,123,113은 PDS에서 사용하는 테이블 이름을 의미합니다.\
